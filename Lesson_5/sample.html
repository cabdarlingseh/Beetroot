Selectors you should know

  1. * - Select ALL
  2. #myId - select all elements with id='myId'
  3. .myClassname - select all the elements with the class='myClassname'
  4. li - select all the elements with the tag "li"
  5. .parent .child - nested selector, select all 'the elements with the class='child' nested inside element with class='parent'
